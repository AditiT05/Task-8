# Task-8
This project is a responsive Simple Blog Website created using Bootstrap 5. The purpose of this task was to understand how to use Bootstrap components such as the navbar, grid system, cards, forms, and footer to build a clean and responsive webpage without writing custom CSS. The website includes a responsive navigation bar, a hero section, a blog section displaying three blog cards, an about section describing the project, and a footer that contains a contact form and social media icons. The layout adjusts automatically for different screen sizes using Bootstrap’s grid system and utility classes. This project helped in understanding responsive design, Bootstrap CDN integration, layout structuring, and the practical use of pre-built UI components to develop modern web pages efficiently.
